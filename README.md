## 📌 Project Overview

A complete, company-level data analytics project analyzing retail customer behavior and shopping trends. This project follows a real corporate workflow — from raw data to executive dashboard — using Python, SQL, and Power BI.

---

## 🎯 Business Problem

A retail company wants to understand customer purchasing behavior to improve sales strategy and retention. Key questions:
- Who are the highest-value customers by segment and category?
- What are the top-performing product categories by revenue?
- How do shopping trends vary by season, age, and gender?
- Which customer segments have the highest repeat purchase rate?

2. **SQL** — Run queries from `sql/` in MySQL / PostgreSQL
3. **Power BI** — Open `.pbix` from `powerbi/` in Power BI Desktop
